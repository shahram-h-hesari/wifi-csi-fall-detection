## Short Project Summary

This repository demonstrates a full research-prototype pipeline for WiFi CSI-inspired fall detection using synthetic signals. The project includes:

- Synthetic CSI-like signal generation and visualization
- Signal preprocessing and statistical feature extraction
- Baseline Random Forest classification
- Clinical-safety-aware metrics: missed falls, false alarms, alarm fatigue
- Adversarial robustness stress testing with three synthetic perturbation types
- Simple preprocessing defense methods with clean/perturbed/defended comparison
- Interactive Streamlit dashboard for visualization and education

All results are prototype outputs using synthetic data. The project is intended to support PhD research, not clinical deployment.
