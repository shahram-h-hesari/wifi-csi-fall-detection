## Dataset Summary

The AntiEave-WiFi-Sensing dataset is associated with the AntiEave-WiFi-Sensing repository (MoWiNG Lab) and the IEEE PerCom 2023 paper *Scheduled Spatial Sensing against Adversarial WiFi Sensing*. The repository includes a dataset folder used for anti-eavesdropping and scheduled spatial sensing experiments.

This dataset appears to support WiFi sensing security / anti-eavesdropping experiments. The exact contents, labels, hardware configuration, number of subjects, and collection environment must be verified from the repository and the associated paper.

> **Important:** This is not a healthcare dataset. This is not a fall-detection dataset. This dataset has not been used for training, validation, or benchmarking in this repository. It is cataloged for literature review and future reproducibility planning only.
