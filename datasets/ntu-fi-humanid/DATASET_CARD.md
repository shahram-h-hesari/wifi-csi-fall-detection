## Dataset Summary

NTU-Fi HumanID (Nanyang Technological University WiFi Human Identification) is a WiFi CSI-based dataset for person identification using WiFi signals, collected at NTU Singapore. It is tracked in this repository because it is included as a supported benchmark dataset in the SenseFi / WiFi CSI Sensing Benchmark.

NTU-Fi HumanID is focused on human identification tasks rather than activity recognition. It is distinct from NTU-Fi HAR (*Pending verification* — confirm dataset separation from original source).

> **Important:** This repository does not claim NTU-Fi HumanID is directly applicable to fall detection. It is cataloged for reference and future benchmarking planning. Applicability to fall detection must be verified from the original dataset paper.
