## Instructions for Adding a Screenshot

1. Run the dashboard locally:
   ```bash
   pip install -r requirements.txt
   streamlit run app.py
   ```

2. Take a screenshot of the dashboard in your browser.

3. Save the screenshot as `figures/dashboard_screenshot.png`.

4. Add it to the repository:
   ```bash
   git add figures/dashboard_screenshot.png
   git commit -m "Add dashboard screenshot"
   git push
   ```

5. Optionally, add it to the README under the Demo Dashboard section:
   ```markdown
   ![Dashboard Screenshot](./figures/dashboard_screenshot.png)
   ```

## Note

Do not add fake or fabricated screenshots. The screenshot should reflect actual dashboard output from a local run of `app.py`.

All dashboard outputs are synthetic prototype results only.
