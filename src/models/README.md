# src/models

This module will contain Python scripts for **baseline ML models** for fall detection classification.

## Planned Contents

- Random Forest baseline classifier (currently in `src/baseline_model.py`)
- CNN-LSTM architecture for temporal CSI classification (Phase 3)
- Stacking ensemble framework (RF + ANN, Phase 3)
- Model evaluation utilities (cross-validation, metrics, confusion matrices)
- Model serialization and loading utilities

## Current Status

The current baseline model is located at `src/baseline_model.py` (flat module). This submodule directory is a forward-looking placeholder.
