# src/features

This module will contain Python scripts for **feature extraction** from preprocessed CSI signals.

## Planned Contents

- Time-domain feature extraction (mean, variance, skewness, kurtosis, energy)
- Frequency-domain feature extraction (FFT-based features)
- Wavelet-based feature extraction
- Sliding window feature computation
- Feature selection and dimensionality reduction utilities

## Current Status

The current feature extraction utilities are located at `src/features.py` (flat module). This submodule directory is a forward-looking placeholder.

> Migration will be done carefully to avoid breaking existing imports.
